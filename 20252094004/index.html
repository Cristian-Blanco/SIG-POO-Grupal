Distancias desde Bogotá

Cómo funciona este mapa

1. Cargar mapa base

Se utiliza Leaflet con un fondo oscuro de Carto para una visualización elegante.

2. Marcar Bogotá

Se coloca un marcador fijo en las coordenadas de Bogotá (Colombia).

3. Cargar países

Se carga un archivo GeoJSON con las fronteras de todos los países.

4. Calcular distancia

Usamos Turf.js para calcular la distancia desde Bogotá al centroide de cada país.

5. Mostrar resultados

Se muestra un popup con la distancia exacta desde Bogotá a cada país.
