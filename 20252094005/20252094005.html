MARICHUELA

Cómo se construyó esta página (paso a paso)

Haz clic en “Ver en la página” para resaltar dónde aplica cada paso.

  1. Paso 1: Estructura base de HTML

    Arrancamos con DOCTYPE, idioma, metaetiquetas y el enlace a estilos.css y programa.js.

    Ver código 
    <!DOCTYPE html>
<html lang="es">
  <head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <link rel="stylesheet" href="estilos.css">
  </head>
  <body>
    ...
    <script src="programa.js"></script>
  </body>
</html>
  2. Paso 2: Encabezado con logo y título

    Un <header> con el logo a la izquierda y el nombre del sitio.

    Ver código 
    <header id="seccion-encabezado">
  <div class="logo">
    <img src="logo.png" alt="Logo">
  </div>
  <h1>Marichuela</h1>
</header>
  3. Paso 3: Menú con submenús

    La barra de navegación usa listas anidadas. En desktop se abre con hover; en móvil, con botón.

    Ver código (HTML) 
    <nav id="seccion-menu">
  <button id="btnMenu">☰ Menú</button>
  <ul class="menu">
    <li class="menu-item">
      <span class="menu-titulo">Equipamientos</span>
      <ul class="submenu">...</ul>
    </li>
    ...
  </ul>
</nav>
  4. Paso 4: Sección “Hero” con overlay

    Una imagen de fondo con un filtro oscuro y un título centrado.

    Ver código (HTML) 
    <section id="seccion-hero" class="hero">
  <div class="overlay">
    <h2>MARICHUELA</h2>
  </div>
</section>
  5. Paso 5: Pie de página fijo

    Barra fija al fondo con los créditos.

    Ver código (HTML) 
    <div id="seccion-footer" class="footer-fixed">
  © <span id="anio"></span> Barrio Marichuela - Todos los derechos reservados.<br>
  Diseño y desarrollo por AKA
</div>
  6. Paso 6: Interactividad con JavaScript

    JS resalta secciones al pulsar “Ver en la página” y maneja el menú móvil.

    Qué hace el JS
    • Resaltar una sección por unos segundos y hacer scroll suave.
    • Menú móvil: abre/cierra el menú y submenús.
    • Año automático en el pie.
