Procedimiento

Para realizar la visualización del polígono se siguio el siguiente procedimiento:

1. Para la creacion de la página se crea el archivo en formato HTML para la visualización del resultado, en donde se cargan todos los aspectos de visualización

2. Se crea el archivo CSS para definir los estilos de la página, incluyendo el mapa y la leyenda

3. Se crea el archivo JavaScript que contiene la lógica para cargar los datos de los parques y el polígono del barrio, así como la configuración del mapa

4. Se utiliza Leaflet para crear un mapa interactivo y cargar los datos de los parques y el polígono del barrio

5. Se define un ícono personalizado para los parques y se añaden marcadores al mapa

6. Se añade un polígono que representa el barrio y se configura su estilo

7. Se añade una leyenda para identificar los elementos del mapa

8. Se asegura que el mapa se ajuste al tamaño de la pantalla y se centra en el barrio

9. Se añade un pie de página con información de derechos reservados

10. Se vinculan los archivos CSS y JavaScript necesarios para el funcionamiento del mapa
