Mapa del Barrio Marichuela - Bogotá

Descripción del Proyecto

En este proyecto se construyó un visor web interactivo utilizando Leaflet y archivos en formato GeoJSON. Se integraron diferentes capas de información geográfica: el polígono que delimita el barrio Marichuela en Bogotá, la red de vías internas y los paraderos del SITP. Los polígonos y líneas se visualizaron directamente a partir de sus archivos GeoJSON, mientras que los paraderos se representaron como marcadores sobre el mapa. Además, a cada marcador se le asoció una fotografía almacenada en el mismo archivo GeoJSON mediante un atributo adicional, de manera que al seleccionar un paradero se despliega su nombre y la imagen correspondiente. Todo este proceso permite consultar de forma sencilla y visual la información espacial y complementaria del barrio, integrando datos vectoriales, atributos y elementos multimedia en una sola aplicación web.
